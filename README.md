# AI-Battleship

## Rules:
  1) Each player arranges ships according to fleet
  2) Take turns firing a shot
  3) Mark Hits and Misses
  4) Call out when a ship has been sunk
  5) Sink all to win

## Ships:
 - Carrier - 5
 - Battleship - 4
 - Cruiser - 3
 - Submarine - 3
 - Destroyer -2

10 Rows x 10 Columns
